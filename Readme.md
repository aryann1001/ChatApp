## üì¶ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/aryann1001/ChatApp.git
   cd ChatApp
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the application**
   ```bash
   npm start
   ```
